# API simulada

## GET /api/health
Retorna status da aplicação.

## GET /api/coletas
Retorna coletas simuladas.

## POST /api/coletas
Cria uma coleta simulada.

## GET /api/rotas
Retorna rota com cálculo de litros, custo, custo por coleta, km por coleta e eficiência.

## GET /api/impacto
Retorna indicadores de impacto.
